# Shreyas Prabhakar's Portfolio

A clean, modern portfolio website built with React and TypeScript, showcasing projects and professional information.

## Features

- Clean, classic black and white design
- Responsive layout for all devices
- Project showcase linked to GitHub repositories
- Contact form for easy communication
- Professional information and skills display

## Tech Stack

- React
- TypeScript
- Styled Components
- Framer Motion for animations
- React Router for navigation
- React Icons for UI elements

## Running Locally

1. Clone the repository
2. Install dependencies with `npm install`
3. Start the development server with `npm start`
4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Project Structure

- `/public` - Static assets
- `/src` - Source files
  - `/components` - Reusable UI components
  - `/pages` - Main page components
  - `/styles` - Global styles and theme

## Deployment

The portfolio is automatically deployed using GitHub Actions to GitHub Pages at [https://shrprabh.github.io/Shreyas_Portfolio/](https://shrprabh.github.io/Shreyas_Portfolio/)

### CI/CD Workflow

This project uses GitHub Actions for continuous integration and deployment:

- When code is pushed to the `main` branch, GitHub Actions automatically builds and deploys the site
- The workflow can also be triggered manually from the Actions tab in GitHub
